# 算力测试参数说明

> 来源: MindCluster 7.3.0 开发文档 - 昇腾社区

---

## 主要参数

| 参数 | 说明 | 是否必填 | 默认值 | 范围 |
|-----|------|---------|-------|------|
| `-f, --flops` | 启用算力测试 | 是 | - | - |
| `-t, --type` | 算子运算类型 | 否 | fp16 | 见下方说明 |
| `-d, --device` | 指定设备 ID | 否 | Device 0 | 芯片逻辑 ID |
| `--et, --execute-times` | 执行次数 | 否 | 60 | [10, 80]（单位：十万次）|
| `--all` | 测试整机算力 | 否 | 单设备 | 不支持与 -d 同时使用 |
| `-q, --quiet` | 跳过确认提示 | 否 | 提示确认 | - |
| `-fmt, --format` | 输出格式 | 否 | normal | normal/json |

---

## 数据类型 (-t) 说明

| 类型 | 说明 | 支持产品 | 矩阵参数 | 向量参数 |
|-----|------|---------|---------|---------|
| `fp16` | 半精度浮点 | 全系列 | m=256, k=32/64, n=128 | n=32760 |
| `int8` | 8位整数 | 全系列 | m=256, k=32/64, n=128 | n=32760 |
| `fp32` | 单精度浮点 | A2/A3 系列 | m=128, k=32, n=64 | n=16380 |
| `hf32` | TensorFloat-32 | A2/A3 系列 | m=256, k=64, n=128 | n=16380 |
| `bf16` | BFloat16 | A2/A3 系列 | m=256, k=64, n=128 | n=16380 |

**说明**：
- A2/A3 系列包含矩阵乘和向量乘计算
- 其他产品仅矩阵乘

---

## 矩阵乘参数

| 数据类型 | m (A矩阵行) | k (A矩阵列/B矩阵行) | n (B矩阵列) |
|---------|------------|-------------------|------------|
| fp16 (所有设备) | 256 | 32 | 128 |
| fp16 (200I/500 A2, 训练系列, A2/A3) | 256 | 64 | 128 |
| int8 (200/300/500 推理) | 256 | 32 | 128 |
| int8 (200I/500 A2, 训练系列, A2/A3) | 256 | 64 | 128 |
| fp32 (A2/A3 系列) | 128 | 32 | 64 |
| hf32/bf16 (A2/A3 系列) | 256 | 64 | 128 |

---

## 向量乘参数

| 数据类型 | 向量长度 n |
|---------|-----------|
| fp16 | 32760 |
| fp32/hf32/bf16 | 16380 |

---

## 执行次数 (--et) 说明

- 单位：十万次
- 范围：[10, 80]
- 默认值：60（即 600 万次）
- 总执行次数 = 单个 AI Core 执行次数 × AI Core 数量
- A2/A3 系列额外包含：单个 Vector Core 执行次数 × Vector Core 数量

---

## 常用参数组合

### 单设备测试

```bash
# 默认 FP16 测试
ascend-dmi -f -q

# 指定设备 0，执行 600 万次
ascend-dmi -f -d 0 --et 60 -q

# INT8 测试
ascend-dmi -f -t int8 -d 0 --et 60 -q
```

### A2/A3 系列专用

```bash
# FP32 测试
ascend-dmi -f -t fp32 -d 0 -q

# HF32 测试
ascend-dmi -f -t hf32 -d 0 -q

# BF16 测试
ascend-dmi -f -t bf16 -d 0 -q
```

### 整机测试

```bash
# 整机 FP16 算力
ascend-dmi -f --all -q
```

## 输出字段

| 字段 | 说明 |
|-----|------|
| Device | 设备 ID |
| Execute Times | 执行次数（总） |
| Duration(ms) | 执行耗时 |
| TFLOPS@<type>/TOPS@<type> | 算力值 |
| Power(W) | 满算力实时功耗 |

---

## 温度要求

- 建议在设备温度稳定并 **低于 90°C** 时进行测试
- 避免温度过高触发主动降频，影响测试结果
