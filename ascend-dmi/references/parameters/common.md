# 公共参数说明

> 来源: MindCluster 7.3.0 开发文档 - 昇腾社区

---

## 全局参数

| 参数 | 说明 | 适用功能 |
|-----|------|---------|
| `-h, --help` | 查看帮助信息 | 所有功能 |
| `-v, --version` | 查看版本信息 | 所有功能 |
| `-fmt, --format` | 指定输出格式 | 所有功能 |
| `-q, --quiet` | 跳过确认提示 | 部分功能 |

---

## 输出格式 (-fmt)

| 格式 | 说明 |
|-----|------|
| `normal` | 正常文本输出（默认） |
| `json` | JSON 格式输出 |

**注意**：
- JSON 格式会保存诊断结果到文件
- 保存路径：`ascend_check/environment_check_before.txt`

---

## 设备指定参数

| 参数 | 说明 | 示例 |
|-----|------|------|
| `-d, --device` | 指定设备 ID（逻辑 ID） | `-d 0` / `-d 0,1,2` |

**获取逻辑 ID**：
```bash
npu-smi info -m
# 查看 Chip Logic ID 列
```

---

## 诊断/压测公共参数

| 参数 | 说明 | 示例 |
|-----|------|------|
| `-r, --result` | 结果保存路径 | `-r /test` |

**默认路径**：
- root 用户：`/var/log/ascend_check`
- 非 root 用户：`$HOME/var/log/ascend_check`

---

## 信息查询参数

### 软硬件兼容性 (-c)

| 参数 | 说明 | 是否必填 |
|-----|------|---------|
| `-p, --path` | CANN 安装路径 | 非 root 或非默认路径时必填 |

**默认路径**：`/usr/local/Ascend`

---

## 眼图测试参数

| 参数 | 说明 | 是否必填 | 默认值 |
|-----|------|---------|-------|
| `--sq` | 启用眼图测试 | 是 | - |
| `-t, --type` | 测试类型 | 否 | pcie |

**测试类型**：
- `pcie`：PCIe 眼图
- `hbm`：HBM 眼图
- `roce`：RoCE 眼图
- `serdes`：Serdes 眼图
- `nic`：NIC 眼图
- `all`：全部类型

---

## NPU 恢复参数

| 参数 | 说明 | 是否必填 | 示例 |
|-----|------|---------|------|
| `-r, --reset` | 启用 NPU 恢复 | 是 | - |
| `-d, --device` | 指定设备 ID | 推荐 | `-d 0,1,2` |
| `-q, --quiet` | 跳过确认 | 否 | `-q` |

**注意**：
- Atlas A2 系列服务器指定 device 时，只复位列表里的第一个 device，表示所有 NPU 复位
- 执行前必须停止 NPU 相关业务
