当前 `skills/ops/` 已承载真实 skill 目录；请按下面链接进入对应 skill 开发：

- [`triton-ascend-migration/`](triton-ascend-migration/)：Triton / PyTorch 算子迁移
- [`npu-op-benchmark/`](npu-op-benchmark/)：算子基准测试与调优辅助

推荐场景：

- 自定义算子开发
- CUDA / Triton 到昇腾迁移
- 算子级性能调优
