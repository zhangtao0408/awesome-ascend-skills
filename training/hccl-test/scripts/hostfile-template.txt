# HCCL Test Hostfile Template
# ============================

# MPICH Format (节点IP:卡数)
# --------------------------
# 单机测试：只写本机 IP 和卡数
# 格式：IP:NPU数量

# 单机 8 卡测试示例（替换为本机实际 IP）：
# 175.99.1.3:8

# 双机 16 卡测试示例：
# 175.99.1.3:8
# 175.99.1.4:8

# 四机 32 卡测试示例：
# 175.99.1.3:8
# 175.99.1.4:8
# 175.99.1.5:8
# 175.99.1.6:8


# Open MPI Format (节点名 slots=卡数)
# ------------------------------------
# 格式：主机名 slots=NPU数量

# 单机测试：
# node1 slots=8

# 双机测试：
# node1 slots=8
# node2 slots=8

# 注意：
# 1. 主机名需要在 /etc/hosts 中配置
# 2. 节点间需要配置 SSH 信任关系
# 3. 属于同一超节点的 AI Server 信息需配置在一起
